# Contributing

Thanks for your interest in contributing! Here's how you can help:

## How to Contribute

1. Fork this repository
2. Create a branch: `git checkout -b add-new-commands`
3. Add your commands/tips in the appropriate section
4. Commit: `git commit -m "Add: description of what you added"`
5. Push and open a Pull Request

## Guidelines

- Keep commands practical and production-tested
- Add comments explaining what each command does
- Use consistent formatting (see existing sections)
- One section per tool/topic

## What to Contribute

- Commands you use daily in production
- Troubleshooting tips
- Better explanations for existing commands
- New tool sections (Helm, ArgoCD, Prometheus, etc.)
- Typo fixes

## Code of Conduct

Be respectful and constructive. We're all here to learn.
