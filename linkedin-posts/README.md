## 💡 Tips for Maximum Engagement

1. **Post between 8–10 AM IST** (weekdays)
2. **Put the repo link in the FIRST COMMENT**, not in the post body
3. **Engage in comments** for the first 30 minutes after posting
4. **Use the carousel** (`linkedin-carousel.html` in repo root) as a visual
5. **Skip weekends** for technical posts (or save lighter content for Sat/Sun)
